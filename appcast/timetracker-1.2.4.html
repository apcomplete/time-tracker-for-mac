Changes in Time Tracker 1.2.4
  • Fixed a bug where projects could not be created the first time Time Tracker was run on a computer.
  • Projects and tasks can now be reordered by dragging and dropping them.
  • Added "Export CSV..." option to export Time Tracker data into Excel. Contributed by Rainer Burgstaller.

Changes in Time Tracker 1.2.3
  • Fixed the problem where project and task names could not be edited when the timer was running. (TT-2)
  • You can now print the entire Time Tracker window (all three panes at once) using the new "Print Window" function, as requested by Rich Wojtas. (TT-24)
  • You can now resize the Projects/Tasks tables relative to each other. (TT-23)
  • Upgraded Sparkle to version 1.5b4. Thanks to Andy Matuschak for maintaining and supporting Sparkle! (TT-19)
  • (refactoring) removed updateTotalTime from TTask and TProject.

Changes in Time Tracker 1.2.2
  • Added auto-updating support via appcasts using Sparkle 1.5b2.

Changes in Time Tracker 1.2.1
  • Re-enables basic print functionality that was present in version 1.1. (Complete printing support will be added in version 1.3.)

Changes in Time Tracker 1.2
  • Fixed occasional crashes when deleting tasks.
  • Added menu items and keyboard shortcuts for starting/stopping the timer and creating new projects and tasks.
  • When adding a new project or task, the new entry is immediately selected for renaming.
  • When selecting a project, the task for that project that we previously selected is reselected.
  • Fixed a bug which allowed the timer to be started when a task was not selected.
