Changes in Time Tracker 1.1
  • Data is now saved to disk every five minutes when the timer is running.
  • Data is now saved when the timer is stopped.
  • Project settings are changed to produce a Universal Binary.
  • Changed the nib file for more efficient use of screen real estate.
  • The window is now resizable.
  • The window's size and position are saved in the system defaults.
  • Ensures that the display is refreshed when the main window is first loaded (this was not the case when the window size/position was restored).

Time Tracker 1.1 has been modified and packaged by Aaron VonderHaar. Time Tracker 1.0 and previous versions were written by Ivan Dramaliev.
